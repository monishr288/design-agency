# Design Agency Homepage

A modern, responsive homepage for a design agency built with React.

## 🚀 Live Demo
[View Live Demo](your-vercel-url-here)

## 📋 Features

- **Hero Section** - Agency name, tagline, CTA buttons with animated gradient
- **Services Section** - 4 service cards with icons and hover effects
- **Portfolio Section** - 6 project grid with overlay animations
- **Contact Section** - Contact form with validation and success message
- **Responsive Design** - Mobile-first approach, works on all devices
- **Smooth Scrolling** - Navigation with smooth scroll to sections
- **Form Validation** - Real-time validation with error messages

## 🛠️ Tech Stack

- React 18
- CSS3 (Flexbox & Grid)
- Lucide React Icons
- React Hooks (useState)
